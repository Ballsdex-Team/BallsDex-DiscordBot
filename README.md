Nationdex is a bot inspired by Ballsdex, where you can collect countryballs on Discord and exchange them with your friends!

You can invite the bot [here](https://discord.com/oauth2/authorize?client_id=1048041508160421888&permissions=3288539856577&redirect_uri=https%3A%2F%2Fwww.nationdexbot.com%2Fbot-invite%2Fbot-auth&scope=bot+applications.commands).
